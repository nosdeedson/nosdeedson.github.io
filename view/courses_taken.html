Curso Spring Boot Ionic

Descrição do curso no site Udemy

Neste curso foi ensinado como fazer um API para vendas online de produtos. Foi criado um fluxo com cadastro e login de usuário, um fluxo completo de navegação de carrinhos de compras, fluxo de fechamento de pedido. Além de uma tela de perfil de usuário com envio de foto pela câmera do celular e armazená-la em um bucket na Amazon.

O Backend foi desenvolvido usando Java com Spring Boot e banco de dados Mysql. Foi usada JPA com Hibernate para mapeamento objeto-relacional, autenticação e autorização com tokens JWT.

O frontend foi desenvolvido em Ionic 3 e Angular 5. Explorou-se a arquitetura MVC, os componentes visuais, reactive forms, navegação, interceptors, local storage; Processo de build para Android.

Curso Sobre Comunicação SOAP

Descrição do curso no site Udemy

Este curso é composto de mais de 4 horas foi abordado como criar um WebService que utiliza a comunicação SOAP, usando WSDL, no curdo usou-se o frameWork Spring Boot, e explicou um pouco sobre os conceitos de XSD, JAXB, WSDL. Foi implantado uma segurança básica através do uso de usuário e senha.

Também foi abordado como criar uma aplicação simples para consumir o serviço. No curso o professor criou uma aplicação Java simples que imprimia no console a resposta, mas no código que foi criado, usou-se o Spring Boot para consumir o serviço, o qual recebia uma requisição, comunicava com o serviço WSDL e respondia a requisição, usando as classes geradas para isto através do WSDL.

WSDL, segundo a Wikipedia, é a abreviação de Web Services Description Language, é uma linguagem baseada em XML usada para descrever serviços Web, que funciona como um contrato, onde é descrito o serviço e como este deve ser acessado e quais métodos estão disponíveis.
