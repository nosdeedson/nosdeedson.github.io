Projetos

Para desenvolver minhas habilidades na linguagem Java desenvolvi um sistema desktop para controle de estoque usando o banco de dados Mysql, projeto criado entre o segundo e terceiro semestre, dando seguimento ao curso superior percebi que devemos tratar os dados inseridos pelos usuários, pois estes podem incluir dados incorretos.

Em um dos empregos que atuei, no setor público, o cálculo do banco de horas era feito manualmente, vi uma oportunidade para desenvolver minhas habilidades em Engenharia de Software, fiz a coleta dos requisitos para entendimento do processo e criei uma aplicação desktop para este cálculo, neste projeto usei a biblioteca iText 5 para gerar PDF com os cálculos realizados pelo sistema.
