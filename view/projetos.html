Início Projetos Cursos Realizados Sobre Mim

Para desenvolver minhas habilidades na linguagem Java deselvolvi um sistema desktop para controle de estoque usando o banco de dados Mysql, dando continuidade nos estudos do curso superior aprendi que eu deveria ter validado os dados inseridos pelo usuário.

Em um dos empregos que atuei, no setor público, o cálculo do banco de horas era feito manualmente, vi uma oportunidade para desenvolver minhas habilidades em Engenharia de Software, fiz a coleta dos requisitos para ententimento do processo e criei uma aplicação desktop para este cálculo, neste projeto usei a biblioteca iText 5 para gerar PDF com os cálculos realizados pelo sistema.

Na página Cursos Realizados é exibido os cursos realizados pela plataforma Udemy, além do(s) curso(s) sendo realizados.
