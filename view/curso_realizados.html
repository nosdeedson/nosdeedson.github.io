Curso Spring Boot Ionic

Descrição do curso no site Udemy

Neste curso foi ensinado como fazer um API para vendas online de produtos. Foi criado um fluxo com cadastro e login de usuário, um fluxo completo de navegação de carrinhos de compras, fluxo de fechamento de pedido. Além de uma tela de perfil de usuário com envio de foto pela câmera do celular e armazená-la em um bucket na Amazon.

O Backend foi desenvolvido usando Java com Spring Boot e banco de dados Mysql. Foi usada JPA com Hibernate para mapeamento objeto-relacional, autenticação e autorização com tokens JWT.

O frontend foi desenvolvido em Ionic 3 e Angular 5. Explorou-se a arquitetura MVC, os componentes visuais, reactive forms, navegação, interceptors, local storage; Processo de build para Android.

Curso Sobre os principais Bandos de Dados

Descrição do curso no site Udemy

Neste curso é composto de mais de 40 horas de videoaulas, em que foram abordados os bancos de dados Mysql, PostgreSQL, SQL Server. O curso foi totalmente prático. Foi demonstrado como usar as linguagens SQL (DQL, DML, DDL), PL/SQL e T-SQL.

Sendo mais específico foi abordado a criação de modelagem conceitual, lógica e física de um banco de dados relacional, como fazer a normalização nas primeira, segunda e terceira formas normais, criar o dicionário de dados do SGBD.

Instalar os SGBD's: MySQL, PostrgreSQL, SQL Server e Oracle, bem como a utilizar as respectivas ferramentas de gerenciamento de bancos de dados.

Realizar consultas utilizando os relacionamentos INNER JOIN, LEFT JOIN, RIGHT JOIN e FULL ALTER JOIN.
Utilizar Triggers, Procedures, Functions e Views.
